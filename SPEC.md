# 旅行予算詳細プランナー【無料】自動作成ツール - 技術仕様書

## 概要

**サービス名**: Travel Budget Planner
**バージョン**: 1.3.0（2026-06-14 M6/M7/M10機能追加）
**更新日**: 2026-06-14
**URL**: https://appadaycreator.com/travel-budget-planner/

交通費・宿泊費・食費・観光費を分類入力して旅行の詳細予算を管理。登録不要・完全無料でご利用いただけます。

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 入力フォームの各項目を確認する
2. 必要な数値・条件をすべて入力する
3. 計算ボタンをクリックして結果を取得する
4. 計算結果と詳細な内訳を確認する
5. 条件を変えて複数パターンを比較する

## よくある質問（FAQ）

**Q: 旅行予算詳細プランナーは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 計算結果は正確ですか？**

参考値としてご活用ください。重要な判断の際は専門家・公式機関にご確認ください。

**Q: 何回でも計算できますか？**

はい、回数制限なく何度でもご利用いただけます。条件を変えて比較にお使いください。

**Q: 入力したデータはどこかに保存されますか？**

いいえ。すべての計算はブラウザ内で処理され、データのサーバー送信は行いません。

**Q: スマートフォンで利用できますか？**

はい、PC・スマートフォン・タブレットすべてに対応しています。


## 機能改善履歴

### v1.3.2（2026-06-14）M7シミュレーション強化版
- **複数パターン一括シミュレーター**: 「バックパッカー」「標準」「ラグジュアリー」3パターンを一度に計算・比較表示
- **条件変更による即時再計算**: シミュレーション実行で複数パターンの結果を同時表示（金額範囲表示）

### v1.3.1（2026-06-14）GSC最適化版
- **FAQPage構造化データ追加**: 5問のよくある質問をFAQPageスキーマで実装（GSC表示回数改善対策）
- **FAQ内容充実**: 2つのFAQセクションに計8問の実践的なQ&Aを追加

### v1.3.0（2026-06-14）M6/M7/M10実装版
- **M6 目標設定・達成度トラッキング**: 目標予算入力で達成度を進捗バー表示
- **M7 複数パターン比較**: 複数回計算で結果を比較表示（差分表示）
- **M10 印刷・結果エクスポート**: 印刷ボタン・クリップボードコピー・CSVダウンロード実装

### v1.2.0（2026-06-05）UX強化版
- **プリセット機能**: 「国内1泊2日」「海外5日間」「高級旅行」の3パターンを用意
  - ワンクリックで入力例が自動入力され、すぐに結果を確認可能
- **計算ボタン状態管理**: 計算実行中は「計算中...」に変更、完了後に元に戻る
- **スムーズなアニメーション**: 結果表示時にスライドイン＆フェードイン効果を追加
- **詳細内訳の表形式化**: テーブル見出し（項目・金額・割合）で視認性向上
- **リセット前の確認**: 誤削除防止のため、リセット実行前に確認ダイアログを表示
- **プリセット説明テキスト**: 「💡 プリセットから選択するか、下に直接入力してください」を表示

### v1.1.0（2026-06-04）
- **4つのカテゴリ入力**: 交通費・宿泊費・食費・観光費を分類入力可能に
- **詳細な内訳表示**: 各カテゴリの金額と全体に占める割合（%）をテーブル形式で表示
- **リセット機能**: 「リセット」ボタンで入力値と結果を一括クリア
- **ローカルストレージ対応**: ページリロード時に前回の入力値を自動復元
- **バリデーション改善**: 1つ以上の金額入力でOK、より柔軟に対応
